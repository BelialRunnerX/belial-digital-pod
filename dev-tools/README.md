# POD Website Development Tools

This folder contains helper scripts and tools to accelerate development while maintaining the iterative, one-feature-at-a-time approach.

## Current Tools

### 1. `generate-product.js`
Quickly generates a new product object in the correct format for `products.json`.

### 2. `deploy-check.sh`
Runs a quick verification after deployment (checks for common issues).

### 3. `feature-template.md`
Template for planning the next small iterative feature.

## Usage
Run tools from the project root. All tools are designed to support fast, focused iterations.
